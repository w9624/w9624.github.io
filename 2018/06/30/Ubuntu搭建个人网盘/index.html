Ubuntu搭建个人网盘

发表于 2018-06-30 | 分类于 Ubuntu

常使用的两个云盘：百度云传视频尽竟然要开会员，坚果云免费的空间太小，所以决定使用Seafile搭建一个云盘

搭建环境

服务器：腾讯云Ubuntu 16.04.1 LTS 64

网盘Seafile：seafile-server_6.2.3_x86-64

安装步骤

  1. 安装mysql环境（要为root用户设置密码），如果已经安装过MySQL，跳过这个步骤

  2. 安装python依赖

  3. 为Seafile创建一个用户,并切换为当前用户

  4. 下载解压seafile（腾讯云提供的下载连接为32位的seafile，在安装配置后期会报错，下面的链接时更改之后没问题的）

  5. 配置seafile


注意： 存储位置 一定要填写当前账户有权限创建以及读写权限的位置，否则会配置失败，用用户名可创建新用户不使用root

  1. 启动seafile（如果配置阶段出错，启动的时候会报异常）

在第一次启动的时候会要求配置管理员账户和密码

  1. 访问
    http://www.yourdomain.com:8000
    http://www.yourip.com:8000
    在第一次访问的时候，不知道为什么页面加载特别慢，ssh连接也没法连接到服务器，我把服务器重启了一下，然后再启动seafile服务就OK了

最后可以在seafile官网上下载，seafile各个平台的软件
