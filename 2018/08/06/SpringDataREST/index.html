Spring Data REST (SprringBoot)

REST即表述性状态传递（Representational State Transfer）：将资源的状态以最合适的形式在客户端和服务器之间转移。
使用上可以简单的理解为：Http定义操作，URL确定资源，从而完成对资源的处理
Spring Data REST 支持将Spring Data JPA等的repository自动转换为REST

两种配置方式：

  1. 继承RepositoryRestMvcConfigration

  2. 修改application.properties
